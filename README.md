# Rails Base

This Rails Base installation contains an extended gemfile. Included are the following gems:

* Postgres
* Dotenv
* Carrierwave
* Mini Magic
* RMagic
* Rspec
* Factory Girl
* FFaker
* Rubocop with extentions
* Rails ERD
* Database Cleaner
* Capybara with extentions
* Shoulda Matchers
* Simplecov
